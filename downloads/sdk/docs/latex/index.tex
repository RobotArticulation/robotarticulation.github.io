 \hypertarget{index_autotoc_md0}{}\doxysection{Mercury\+SDK}\label{index_autotoc_md0}
This is the repository for the software development kit (SDK) for the Mercury range of digital servos

Please note that this SDK is based on the Dynamixel SDK. Please see the copyright notices in the source file headers.

\doxysubsubsection*{\uline{Installation}}

Clone the \href{https://github.com/RobotArticulation/MercurySDK}{\texttt{ Mercury\+SDK}}\hypertarget{index_autotoc_md1}{}\doxysubsubsection{Build and install the Mercury\+SDK\+:}\label{index_autotoc_md1}
\hypertarget{index_autotoc_md2}{}\doxyparagraph{Linux (x64)}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item cd Mercury\+SDK/c++/build/linux64
\item make
\item sudo make install
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxyparagraph{Windows (x64)}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item in Visual Studio, open the solution file Mercury\+SDK/c++/build/win64/mcy\+\_\+x64\+\_\+cpp.sln
\item build-\/all in the release configuration.
\end{DoxyItemize}

\doxysubsubsection*{\uline{Running the examples}}\hypertarget{index_autotoc_md4}{}\doxysubsubsection{Building and running the ping example\+:}\label{index_autotoc_md4}

\item build-\/all in the release configuration.
\item ensure that mcy\+\_\+x64\+\_\+cpp.\+dll is in the PATH
